# Data-Structures-and-Algorithms
A collection of Data Structures and Algorithms projects completed through independent practice.

This repository demonstrates the implementation and analysis of fundamental and advanced data structures, algorithmic problem-solving techniques, and computational efficiency concepts using Java.

---

## Topics Covered

### Data Structures
- Binary Search Trees (BST)
- Heaps
- Priority Queues
- Hash Tables
- Trees
- Recursion
- Sorting Algorithms

### Graph Theory
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Directed Acyclic Graphs (DAGs)
- Topological Sorting
- Dijkstra's Algorithm
- Minimum Spanning Trees
- Kruskal's Algorithm

### Algorithm Design Techniques
- Divide and Conquer
- Greedy Algorithms
- Dynamic Programming
- Backtracking

### Graph Problems
- Euler Paths
- Euler Circuits
- Hamiltonian Paths
- Hamiltonian Circuits

### Optimization Problems
- Fractional Knapsack
- 0/1 Knapsack

---
